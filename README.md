## Usage

```js
git clone https://github.com/goriunov/vuejs-rollup-boilerplate.git
cd vuejs-rollup-boilerplate
npm install
npm start
```

Thanks to [vue-rollup-boilerplate repo](https://github.com/dangvanthanh/vue-rollup-boilerplate).
